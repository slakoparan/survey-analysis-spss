# Survey Data Analysis in SPSS | SPSS ile Anket Verisi Analizi

This project includes basic statistical analysis of a small survey dataset using IBM SPSS.  
Proje, SPSS kullanılarak yapılan küçük bir anket verisinin temel istatistiksel analizlerini içermektedir.

## Analyses Included / Yapılan Analizler:
- Frequency Tables / Frekans Tablosu
- Bar Chart Visualization / Çubuk Grafik
- Crosstab Analysis / Çapraz Tablo
- T-Test
- ANOVA
- Correlation Analysis / Korelasyon
- Effect Size (Cohen's d, eta-squared etc.)

## Tools:
- IBM SPSS Statistics
